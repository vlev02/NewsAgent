### 🔥 今日必看

- **ChronoStitch: Training-Free Composition of Visual KV Memories for Long-Horizon Temporal Reasoning**（arXiv 2607.19547，7/21）— 独立缓存的视频分块 KV 各自从局部 RoPE 位置 0 开始 prefill，朴素拼接会造成时间相位冲突、丢失"先后顺序/次数/变化"等全局时序结构；提出训练-free 方法协调跨块位置编码来正确缝合视觉 KV memory。继 Kamera、LazyAttention、C²KV 之后又一次呼应我们反复追踪的 R1（多块拼接是否 scramble 位置）问题，且这次证据来自视觉/长时序 QA 场景而非纯文本，说明"位置墙"不是纯文本 LLM 特有的问题。
- **Twin Agent: Context Residual Compression for Privilege Separated Agents**（arXiv 2607.19595，7/21）— 面向 agent 因不可信 observation 触发 prompt injection、操纵下游推理/工具调用的安全风险，提出 Explore Agent（接触不可信信息）/ Safe Agent（执行特权动作）对称双 agent 设计，Safe Agent 只接收压缩后的 residual hint、不直接看到原始不可信 context；是 ReAct 的即插即换替代方案。把"隔离不可信 context"和"控制传给特权执行链路的信息量"结合成同一个 context 管理问题，是 tier_2 里一个新的架构变体，而非传统压缩/驱逐路线。
