### 🔥 今日必看

- **MosaicKV**（arXiv 2607.00760）— serving 场景下的动态二维 KV cache 压缩方案，利用 GPU/CPU 协同维护压缩后的 cache 并加速 attention 计算：相比未压缩基线 attention 最高 16x 加速、decode 延迟降至 1/4.8、吞吐提升 7.3x、显存降至 1/3，LongBench/RULER 上平均精度损失仅 1.76%。是今天 tier_0/tier_1 交集里结果最扎实的一篇。
- **Governance Decay: How Context Compaction Silently Erases Safety Constraints in Long-Horizon LLM Agents**（arXiv 2606.22528）— 提出 ConstraintRot 基准，测了七个模型家族：完整 context 下 agent 违反安全约束的比例是 0%，但经过一次 context compaction 后违规率最高升到 30%。直接挑战"context compaction/摘要只是有损但安全无关"的默认假设——对我们关注的 agentic KV/context 管理来说，这意味着压缩策略需要显式识别并保护 safety-critical 内容，而不能纯按 recency/attention mass 驱逐。

### 📄 新论文

#### KV Cache 压缩/驱逐
- **MosaicKV: Serving Long-Context LLM with Dynamic Two-D KV Cache Compression**（2607.00760）— 见上，tier_0+tier_1 双料命中。
- **IntentKV: Cross-Turn Intent-Aware KV Cache Pruning for Agent Inference**（2606.09916）— watchlist 项目首次搜到公开论文：冻结基座 LLM，维护 session 级 QueryMemory 记录跨轮意图，用 memory-attention 规则给历史 token 打分做剪枝，针对多轮 agent 场景"短查询展开成长轨迹、KV cache 成为主要 serving 瓶颈"的问题。

#### KV Cache 复用 / Position Re-indexing
- **Cache Merging as a Convergent Replicated State for Multi-Agent Latent Reasoning**（2607.01308）— 多智能体 latent reasoning 把各 agent 的 KV cache 拼接成单一 context 供最终 agent 使用；指出既有做法 BagMerge（沿序列轴拼接 + RoPE 重编码）不满足交换律、最优输入顺序不可预测。提出 **CanonicalMerge**：按中间层 mean K-norm 对缓存重排序，使合并结果与输入排列无关（byte-identical），在效果上与最优 BagMerge 排序打平，但不需要事先知道哪种顺序最优。BagMerge/CanonicalMerge 是本文自造的新术语，值得后续追踪。

### 🗑️ 过滤掉的

- Vision token merging/pruning 相关（RAPID、AdaMerge、LearnPruner、MeToM/EarlyTom 等）— 均为 ViT/VLM 视觉 token 层面的裁剪/合并，未落回 LLM 的 KV cache 算法或系统层面，按 deprioritize 规则过滤。
- Attention sink 综述与机制类论文（Attention Sink Survey 2604.10098、A Mechanistic Account of Attention Sinks in GPT-2 2604.14722、Sink vs. diagonal patterns 2605.08453 等）— 都是本周之前已发布的解释性/综述工作，本次搜索只是被重新索引到，没有和 KV cache 压缩/驱逐/复用直接挂钩的新方法，不计入新论文。
- Speculative decoding 相关新检索结果（BudgetDraft、CXL-SpecKV、KNN-SSD 等）— 大多聚焦 draft/verify 效率本身，KV cache 只是背景设定而非核心贡献，按 deprioritize 中"和 tier_0/1 无交集"过滤；DSpark（2607.05147）因确实涉及 KV cache 管理已在 7/16 收录，不重复。
- CacheClip、SideQuest、Attention Matching（Fast KV Compaction）等 watchlist 项目本次搜索命中的仍是此前已知版本（2510.10129 / 2602.22603 / 2602.16284），未发现新更新，跳过。
